# 森音チンチラカフェ UI実装計画書

## 目的

生成済みの日本向けUIモック画像を参照し、React + Tailwind CSSで架空の高級チンチラカフェサイトを実装する。
目標は、画面構成、余白、色、文字、写真の扱い、ボタン、カード、表、予約UIをできるだけ参照画像に近づけること。

## 技術前提

- React + Vite
- Tailwind CSS
- ルーティングは軽量化のため、React stateでページ切り替えする
- 画像は `/public/assets` に配置する
- 参照用モック画像は `/public/references` に残す
- 外部フォント取得に依存しない。CSSのフォントスタックで明朝系とゴシック系を指定する
- アニメーションはCSSと少量のReact stateで実装する

## ページ構成

1. トップ
   - ヘッダー、ヒーロー、3つの導入ブロック
   - CTA: `予約する`, `空間を見る`
2. チンチラたち
   - ヒーロー、4匹のプロフィール、週間出勤予定
   - CTA: `出勤予定を見る`, `予約する`
3. 空間
   - ヒーロー、`座席`, `照明`, `音環境`, `温度と空気`
   - `香り` は出さない
4. メニュー
   - ヒーロー、`季節のお茶（人用）`, `焼き菓子（人用）`, `チンチラのおやつ`, `チンチラカフェコース`
   - `軽食` は出さない
5. 初めての方へ
   - ヒーロー、利用の流れ、お願い、よくある質問、CTA
6. ご予約
   - カレンダー、時間、人数、コース、利用時間、備考、予約内容、キャンセルポリシー

## UI方針

- 参照画像に合わせ、上部ヘッダーは白寄りのリネン背景、ロゴ左、ナビ中央、予約ボタン右
- 見出しは大きい明朝系、本文は読みやすい日本語ゴシック系
- ボタンは濃いモスグリーンと枠線ボタンの2種類
- 角丸は控えめにし、カードは `8px` 前後
- 境界線は薄いブラウン系
- 写真は暖色の木目、自然光、チンチラ、陶器、真鍮の雰囲気にそろえる

## アニメーション方針

- 初回表示時にヒーローの文字と写真をゆっくりフェードアップ
- ページ切り替え時にコンテンツを短くフェード
- カードはホバー時に少しだけ浮かせる
- ボタンはホバー時に矢印が少し右へ動く
- 予約フォームの選択状態は滑らかに色を変える
- `prefers-reduced-motion` では大きな動きを止める
